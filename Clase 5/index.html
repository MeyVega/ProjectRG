Animaciones

Las animaciones permiten movimientos más avanzados y se crean con @keyframes y animation

Transiciones

Una transición necesita la propiedad que cambiará y el tiempo de duración (como mínimo)

Movimiento en la web

Las animaciones y transiciones en las páginas web permiten que los elementos cambien de estado de manera fluida. Pueden ser simples (como un color que cambia) o complejas (como algo que se mueve)
